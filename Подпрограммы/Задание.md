## 2.2 (1 балл)
### Перед вами функция поиска НОД двух чисел:
```cpp
// Функция поиска наибольшего делителя положительных чисел a и b
int getMaxCommonDivider(int a, int b) {
    int commonDivider = 1;
    int minNumber;
    
    if (a > b)
        minNumber = b;
    else
        minNumber = a;

    for (int i = 1; i <= minNumber; i++)
        if ((a % i == 0) && (b % i ==0))
            commonDivider = i;

    return commonDivider;
}
```
